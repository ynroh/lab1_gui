1 Постановка задачи

Функция – это фрагмент кода, который выполняет определенную задачу или операцию.

Формат записи:

"fun имя_функции(аргумент1: тип_аргумента1, аргумент2: тип_аргумента2): тип_функции{
         return аргумент1 + аргумент2
}"

Пример:

fun sum(a : Int, b: Int): Int { 
    return a + b 
}

В связи с разработанной автоматной грамматикой G[‹Def›] синтаксический анализатор (парсер) десятичных констант будет считать верными следующие записи функций:

  1. fun sum(a : Int, b: Double): Double{ 
    return a + b 
}
            
  2. fun sum(a : Int, b: Int, c: Int): Int { 
    return a + b + c
}
            
  3. fun sum(a : Int, b: Int): Int { 
    return a * b 
}
