2 Разработка грамматики

Определим грамматику функций языка Kotlin G[‹Def›] в нотации Хомского с продукциями P:

DEF → ‘fun’ SPACE 
SPACE → ‘ ‘ NAME
NAME →  Б { Б | Ц | ‘_’ }  ‘(‘ OPENARG 
OPENARG → ARGUMENT
ARGUMENT →  Б { Б | Ц | ‘_’ } ‘:’ TYPEARG 
TYPEARG →  ‘Int’ | ‘Double’ | ‘Long’ | ‘Float’ COMMA | COLON
COMMA  →  ‘,’ OPENARG 
COLON →  ‘)’ ‘:’ TYPEFUN
TYPEFUN → ‘Int’ | ‘Double’ | ‘Long’ | ‘Float’ OPENFUNC
OPENFUNC → ‘{‘ RETURN
RETURN →  ‘return’  EXPR
EXPR → ‘ ‘ ARG1
ARG1 →  Б { Б | Ц | ‘_’ } SIGN 
SIGN →  ‘+’ | ‘-‘ | ‘*’ | ‘/’ ARG2
ARG2 →  Б { Б | Ц | ‘_’  ENDBODY | SIGN 
ENDBODY→  ‘}’ END
END →  ‘;’
Б → ‘a’ | ‘b’ | … | ‘z’ | ‘A’ | ‘B’ | … | ‘Z’
Ц → ‘0’ | ‘1’ | … | ‘9’

Следуя введенному формальному определению грамматики, представим G[‹Def›] ее составляющими:

Z = ‹Def›;
VТ = {‘ fun’ ‘return’ ‘Int’ ‘Double’ ‘Long’ ‘Float’ ‘a’…’z’  ‘A’…’Z’ ‘0’…’9’ ‘(‘ ‘ ‘ ‘:’ ‘,’ ‘)’ ‘{‘ ‘_’ ‘+’  ‘-‘  ‘*’  ‘/’  ‘}’ }
VN = { DEF, SPACE, NAME,  OPENARG, ARGUMENT, TYPEARG , COMMA ,COLON, TYPEFUN,  OPENFUNC, RETURN, EXPR, ARG1, SIGN , ARG2, ENDBODY,  END,  Б, Ц}
